# Endogenous Reachability Collapse (ERC)

A minimal dynamical framework for understanding loss of recoverability without removal of state space.

---

## 📌 Overview

This repository presents a sequence of minimal models exploring how systems can lose functional recoverability due to internal constraints and temporal dynamics.

The framework introduces the concept of **reachability collapse**, where:

> The attractor remains in the system, but becomes dynamically inaccessible.

---

## 📚 Structure

The project is organized into three complementary papers:

### 📄 Paper 1 – Minimal ERC Model
- Introduces the core dynamical system
- Demonstrates loss of recoverability via contraction of accessible state space

📁 `paper1/`

---

### 📄 Paper 2 – Ensemble Reachability Collapse
- Extends the analysis to distributions of initial conditions
- Defines recoverability as a measurable fraction

📁 `paper2/`

---

### 📄 Paper 3 – Rate-dependent Collapse
- Introduces time-dependent forcing
- Shows collapse depends on input rate, not only magnitude

📁 `paper3/`

---

## 🧠 Core Insight

Across all three papers:

- Loss of function may occur **without loss of structure**
- Collapse emerges from:
  - internal constraint accumulation
  - shrinking accessibility
  - mismatch of timescales

These are minimal dynamical models.
